System Setup

Welcome to ZION. Before you can use the live dashboard, Trend-Lock, and market data tools, you need to connect your data source. This takes about five minutes. The education modules (Academy, Primer, all Modules) work immediately with no setup required.

01 Market Data API Key Required for live tools

The ZION Dashboard, Trend-Lock, and live screeners pull real market data from Polygon.io (also called Massive). You need a Polygon.io account and API key to use these tools. The education modules work without it.

1.Go to polygon.io/dashboard/signup and create a free account
2.Choose the Starter plan (~$29/month) — this gives you the data ZION needs
3.In your Polygon dashboard, go to Keys and copy your API key
4.Paste it below and click Save — ZION stores it locally in your browser, never on a server

02 Your Watchlist Customize

The Dashboard loads a default watchlist of tickers. Replace them with the names you actually trade. One ticker per line.

Your Watchlist (one ticker per line)
These tickers will load automatically in the Dashboard. You can also add/remove them directly in the Dashboard at any time.
03 Trade Tracker Optional — starts empty

The Trade Tracker starts completely empty — ready for your first entry. Use natural language to add positions: "Bought 2 NVDA $900 calls exp Jan 17 at $12.50"

All trade data is stored in your browser's local storage — it never leaves your device. If you clear your browser data, your trades will be cleared too. Export your trades regularly using the Export button in the Trade Tracker.

Privacy note: ZION stores zero user data on any server. Your API key, watchlist, trades, and journal entries all live in your browser. No accounts, no cloud sync, no data collection.
04 Command Center Ready to use

The Command Center is your morning launch page. Open this first every trading day. It shows:

◆Live ET clock with current trading session window (Pro Setup, Midday Chop, Smart Money Return, etc.)
◆Macro futures strip — NQ, ES, YM, RTY, VX, DX — click any tile for live Yahoo Finance data
◆Quick links to every ZION tool, external platform, and reference
◆Open positions count and cost basis (pulled from Trade Tracker)
Bookmark ZION_Command.html as your browser homepage or new tab page. Open it every morning before you open anything else.
