Common Questions

FAQ

Do I need to know Ichimoku to start? +
No. The free Primer includes a full Ichimoku introduction that covers all five components before Module 2 goes deep. You'll build the system piece by piece — including an interactive chart that lets you add each component yourself and watch it draw in real time.
What platform do I need? +
TradingView for charting (free account works), a brokerage for execution (Robinhood, Fidelity, TD Ameritrade — your choice), and the Massive API (Polygon.io) for the live dashboard (~$29/month). The course modules work in any browser with no additional software.
How much screen time does ZION require? +
1-2 hours per day for active trading. The morning read takes 20-30 minutes. The Pro Setup Window (10:00-11:30am ET) is the primary entry window. Structure trades develop over days — you're not watching every tick. The system is designed for people who have lives.
Is this for day trading or swing trading? +
Structure trading — which sits between swing and position trading. Typical holds are days to weeks. ZION uses 65-minute and Daily timeframes for signal reads, which filters out intraday noise and favors options with 30-90 days to expiration. This is not a 0DTE system.
What if I'm already a 0DTE or momentum trader? +
ZION respects every trading style. Some of the best traders around use 0DTE and momentum strategies — it requires real skill and works for the right operator. ZION is a different path for a different operator. The two can co-exist. Many ZION traders use the structure read to inform all their decisions, regardless of timeframe.
